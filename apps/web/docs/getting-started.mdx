# Getting Started with Cortex

Cortex is a modular desktop markdown editor built for writers and developers who want full control over their knowledge base. It combines a distraction-free editing experience with a powerful plugin system.

## Installation

Download the latest release for your platform from the releases page. Cortex runs natively on macOS, Windows, and Linux via Tauri.

### System Requirements

| Platform | Minimum Version |
|----------|----------------|
| macOS    | 12 Monterey     |
| Windows  | 10 (64-bit)     |
| Linux    | Ubuntu 22.04+   |

## Opening a Vault

A **vault** is a folder on your filesystem that Cortex treats as a workspace. All your notes, attachments, and configuration live inside it.

1. Launch Cortex
2. Click **Open Vault** or press `Cmd+O`
3. Select any folder on your system
4. Cortex will scan it and build your note graph

> Cortex never locks you in. Your files are plain `.md` files — readable by any text editor.

## Writing Your First Note

Create a new file with `Cmd+N`. The editor supports full CommonMark markdown with extensions:

```markdown
# My First Note

This is a paragraph with **bold** and _italic_ text.

## Links between notes

You can link to other notes using [[wiki-links]].
Tags work too: #idea #project

## Code blocks

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`
}
```
```

## Keyboard Shortcuts

| Action           | macOS         | Windows/Linux   |
|------------------|---------------|-----------------|
| New note         | `Cmd+N`       | `Ctrl+N`        |
| Open vault       | `Cmd+O`       | `Ctrl+O`        |
| Quick switcher   | `Cmd+P`       | `Ctrl+P`        |
| Toggle preview   | `Cmd+Shift+P` | `Ctrl+Shift+P`  |
| Split pane       | `Cmd+\`       | `Ctrl+\`        |

## Themes

Cortex ships with two built-in themes:

- **Paper** — A warm light theme inspired by physical notebooks
- **Ink** — A dark theme with amber accents for late-night writing

Switch themes via the status bar at the bottom of the window.

## Next Steps

Once you have your vault set up, you can extend Cortex with plugins. Head over to [First Plugin](/docs/first-plugin) to learn how to build your own.
