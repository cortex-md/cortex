Fundamentos do Cortex
Arquitetura local-first

O Cortex armazena todas as notas como arquivos .md no disco. A sincronização é opcional e nunca bloqueia o fluxo de trabalho. Cada vault tem um UUID permanente que sobrevive a movimentações de pasta.

O espaço em branco não é um erro — é o que separa uma nota boa de uma nota legível.

Textos com destaque em malva e links internos como [[Design System]] são renderizados pelo Live Preview sem recarregar a página.

Código e configurações

Acesse as configurações via Ctrl+, ou pelo Command Palette com Ctrl+P.
